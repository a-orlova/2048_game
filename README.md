# 2048 game

https://2048game1-weld.vercel.app/

2048 - игра, цель которой управлять игровым полем, объединять числа одинакового значения и достигнуть ячейки с числом 2048.

в проекте есть возможность вводить имя игрока после завершения игры для сохранения результата, присутствует таблица лидеров с топ-10 лучших сессий, возможность отмены последнего действия, поддержка клавиш управления и мобильного интерфейса

игра полностью динамическая: все игровые ячейки создаются с помощью js, без использования innerHTML и outerHTML
